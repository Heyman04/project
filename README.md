# Personal Portfolio Website

## Overview

This is a responsive **personal portfolio website** built using **HTML, CSS, and JavaScript**. The website showcases my profile, technical skills, education, and development projects in a clean and modern layout. It serves as a professional platform where visitors and recruiters can explore my work and learn more about my background.

## Features

* Responsive design for desktop and mobile devices
* Modern and clean user interface
* Smooth navigation between sections
* Project showcase section
* Skills and education display
* Contact section for communication

## Technologies Used

* HTML5
* CSS3
* JavaScript
